User Stroy #3

As a logged in user, the user want to watch a popular TV shows or movies which other people on Twitter discuss the most. The user will open up the application and sign in using twitter account. After logged in, the user will see a selection as "Popular" and tap on that. It will lead the user to a new page where defines different categories about shows and movies they can select from. These movies or shows are the most frequently discussed shows or movies on Twitter by Twitter users. If the user wants to watch one of the movies or shows, he can choose Link to Netflix and watch it.
